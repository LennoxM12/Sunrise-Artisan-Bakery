# Interactive Website - POE Project

## Description

This is a comprehensive interactive website built for POE submission, featuring various JavaScript elements, a Google Map, gallery lightbox, dynamic content, search functionality, SEO optimizations, and contact/enquiry forms with validation and email processing.

## Structure

- `index.html`: Main HTML file with all sections.
- `css/style.css`: Stylesheet for the website.
- `js/script.js`: JavaScript file containing all interactive functionalities.
- `images/`: Directory for image assets.

## How to Use

1. Open `index.html` in a web browser.
2. Navigate through the sections using the menu.
3. Interact with accordions, tabs, modal, gallery, map, and forms.

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Features

- Interactive accordions and tabs
- Modal windows
- Gallery with lightbox
- Google Maps integration
- Real-time dynamic content (clock)
- Search functionality
- Contact and enquiry forms with validation and email sending
- SEO optimized with title and meta tags
- Proper image alt texts
